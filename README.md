# My Todo List App

A simple todo list built with React + Vite, styled with Bootstrap.

## Features
- Add, complete, and delete tasks
- Separate views for pending and completed tasks
- Scrollable task lists

## Run locally
\`\`\`bash
npm install
npm run dev
\`\`\`
